## Featured Projects

### Retail Sales Intelligence Platform
End-to-end AWS pipeline ingesting multi-source retail data, processed via Glue and PySpark, modeled into a star-schema Redshift warehouse and surfaced through Power BI dashboards tracking sales KPIs, inventory, and customer behaviour.

`AWS Glue` `Amazon S3` `Amazon Redshift` `PySpark` `Power BI`

---

### Financial Fraud Detection & Risk Monitoring System
Transaction monitoring system using Kinesis Streams and ML models to flag anomalous activity in real time. Features automated alerting via Lambda, audit logging in S3, and a compliance-facing Power BI risk dashboard.

`Amazon Kinesis` `AWS Lambda` `PySpark` `SQL` `Python` `Power BI`

---

### AI Healthcare Recommendation Engine
Lakehouse platform on AWS unifying patient records, diagnostics, and treatment history to drive an ML-based disease prediction and personalised recommendation engine. Built on S3 + Glue with a medallion architecture for data governance and quality.

`Python` `Amazon S3` `AWS Glue` `Apache Spark` `SQL`

---

### Customer Behaviour Analytics Dashboard
360° customer analytics solution consolidating CRM, transactional, and behavioural data into a unified semantic model on Redshift. Delivers cohort analysis, churn signals, and executive reporting via Power BI.

`SQL` `Amazon Redshift` `AWS Glue` `Power BI` `Python`
